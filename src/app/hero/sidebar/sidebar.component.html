Side Bar

Using Anglar Metarial Build Angular Side Bar.

Description

  • To use angular material add angular material to current project ng add @angular/material And Import Mat Component in app.modules.ts file
  • And angular material has component to develop the menu list
    • mat-sidenav-container : acts as container for side nav and content
    • mat-sidenav-content : represents main contents(and routing path can be loadded here)
    • mat-sidenav : repesents the side content (menu list ) , by using #sidenav we can toggle the menu and also to make side moment of menu "mode=side" and to keep open for default use opened="ture"
    • mat-nav-list : use to list the all menu items using mat-item-list
    • mat-toolbar : to design toolbar it uses mat-toolbar-row
    • mat-icon-button & mat-icon: to design button
