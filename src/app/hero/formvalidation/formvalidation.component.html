Description

Forms are used in various seniors of buissiness applictaion.for registarion.login,update,etc
We have two kind of forms in angular Reactive forms and Templete Driven Forms

  • Data Flow In Reactive
    • View To Model : data folw from view to model using formControl for every input feild.
    • Model To View : we can set data to view using setValue (its manditory to set all the feilds in form)or PatchValue (To set only some particular feild of form )
  • to use forms the very first thing import reative forms in app.module.ts file,and also in component.ts file import FormControl,FormBuilder of Form etc
  • And angular forms have
    • form-control : which takes care of input feild , what ever the input user enters that get store in model using fromControlName having multiple form controler which all togther helps
    • form-group : which is collection of formcontrol , which has control over a from-control.
    • ngModel : is use to directly bind the value from both view and model.
    • ngSubmit: which submit the form with all the data of forms.
    • we have load the form value before we submit the form and then after we can add submit forms by appending the values.
